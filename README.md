## Génération du CSS

Pour générer le fichier CSS, utilisez les commandes suivantes :

- Pour construire le CSS une seule fois :
  ```bash
  npm run build:css
  ```

- Pour surveiller et construire automatiquement lors des changements :
  ```bash
  npm run watch:css
  ```

Assurez-vous que `npx` est disponible et que vous avez installé les dépendances nécessaires via `npm install`.
